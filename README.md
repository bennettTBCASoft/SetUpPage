# SetUpPage

### Feature
- Button可以做切換兩個頁面的功能
- 有PickerView的功能
- Reset的Button
- Launch的Button會進到Home Page
- 切換頁面時平面翻轉
- 按鈕圓角、框線顏色、框線寬度、加入底線
- IBDesignable & IBInspectable
